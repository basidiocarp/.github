# Archived Campaigns

This folder holds finished multi-file campaigns that no longer belong in the active `.handoffs/campaigns/` area.

Use these as historical context, source material for new standards, or evidence for follow-up work. Do not dispatch directly from them without first either:

- copying the needed material back into an active campaign, or
- creating a fresh repo-owned handoff that links to the archived campaign
